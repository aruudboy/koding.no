## Ansvar

Som en frontend utvikler er du ansvarlig for å lage det som brukeren ser og interagerer med.

Noen av tingene en frontend utvikler må tenke på er:

- Nettsiden skal være tilgjengelig for alle, uavhengig av funksjonsevne
- Nettsiden skal fungere på alle skjermstørrelser
- Nettsiden skal være rask å laste inn, og enkelt søkbar for søkemotorer
- Nettsiden skal være GDPR-vennlig
- Bruker-erfaringen skal være god og intuitiv

I praksis betyr dette at du må kunne HTML, CSS og JavaScript, og ha en forståelse over hvordan en nettside henter og lagrer informasjon på internett. I tillegg er det nyttig å kunne et rammeverk som React, Vue eller Angular.

## Server og klient

En frontend utvikler sin kode kjøres på brukerens datamaskin (_klienten_), og snakker med en _server_ via HTTP-protokollen. Dette er en viktig forskjell fra backend utvikling, som kjører på _serveren_.

Dette betyr at du som frontend utvikler må kunne litt om hvordan en server fungerer, og hvordan du kan snakke med den via HTTP-protokollen.
