# Opplæring

## Fremgangsmetode

På fagområdene sine sider finner du en liste med mål. Disse kan du bruke som en guide for å lære deg ditt fagomåde. **Du søker selv opp ressurser som kan hjelpe deg med å nå målene**.

Når du har nådd et mål, kan du huke av målet i listen.

Hvis du blir ferdig med alle målene, kan du gå til [veien videre](../veien-videre). Husk at du kan bruke [ressursene](/wiki/ressurser) som finnes på wikien her for å hjelpe deg.
Synes du at det er vanskelig, kan det være lurt å ta en gjennomgang på [selvopplæring og motivasjon](../selvopplaering-og-motivasjon).
