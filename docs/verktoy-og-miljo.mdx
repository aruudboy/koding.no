# Verktøy & Utviklingsmiljø

Når man jobber med koding, så er det generelt sett på en datamaskin. Det fins mye programvare du kan installere på din datamaskin for å gjøre det enklere å skrive kode.

## Operativsystem

Generelt sett, så vil det gå fint å jobbe i både Windows, Mac og Linux baserte operativ systemer.

Det finnes unntak, så søk litt rundt på nettet for å finne ut om det er noen spesifikke krav for det du skal jobbe med.

## Essensielle verktøy

### Kode editor

[Visual Studio Code](https://code.visualstudio.com/) er brukt for å skrive kode, og er tilgjengelig på alle store operativ systemer. VSCode er veldig populært og har derfor også et aktivt extension-miljø. Du kan også såklart [bruke en annen kode editor](https://google.com/search?q=Top%20code%20editors%202023) hvis du ikke liker VSCode.

### Git

[Git](https://git-scm.com/) er et versjonerings-system. Det blir brukt for å lagre en historie av versjoner av koden din. Dette er industri-standard, og blir brukt innenfor alle fagområder som inneholder koding. Kode som er versjonert med git blir ofte lagret på [Github](https://github.com/) eller [Gitlab](https://gitlab.com/).
